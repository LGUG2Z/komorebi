# active-window-border-colour

```
Set the colour for the active window border

Usage: komorebic.exe active-window-border-colour [OPTIONS] <R> <G> <B>

Arguments:
  <R>
          Red

  <G>
          Green

  <B>
          Blue

Options:
  -w, --window-kind <WINDOW_KIND>
          [default: single]
          [possible values: single, stack, monocle]

  -h, --help
          Print help

```
